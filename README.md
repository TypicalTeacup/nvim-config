### Linux/macOS instalation
```sh
git clone https://github.com/TypicalTeacup/nvim-config.git ~/.config/nvim
```

### Windows installation
```powershell
git clone https://github.com/TypicalTeacup/nvim-config.git $env:LOCALAPPDATA\nvim
```
